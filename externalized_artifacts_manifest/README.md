# Externalized Artifacts Manifest

Raw SailGP data is not committed to this repository and should be obtained from the official challenge source under its applicable terms.

Large intermediate artifacts, processed tables, model files, caches, and research outputs are externalized and are not included in GitHub. The small curated derived demo assets in `apps/web/public/foilbrief-data/` are included because the judge-ready web app uses them at runtime.

The Python pipeline is intentionally not included in this clean app-only repository. Full raw-data reproduction requires external data access and a future tested dependency manifest. Private local filesystem paths are not public download links and must not be presented as such.
